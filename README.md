# Email-Fraud-Detection

## Introduction
This project is to detect email frauds using machine learning algorithms. The dataset is from [Kaggle](https://www.kaggle.com/rtatman/fraudulent-email-corpus). The dataset contains 3,000 emails with 1,500 being legitimate and 1,500 being fraudulent. The fraudulent emails are phishing emails that are trying to get the user to click on a link and enter information. The legitimate emails are from Enron employees. The goal of this project is to build a model that can detect fraudulent emails.
